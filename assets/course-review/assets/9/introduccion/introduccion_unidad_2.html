En esta unidad podrás conocer los conceptos asociados a la creación de bases de datos documentales haciendo uso de un sistema de gestión de bases de datos no estructuradas.

De igual forma, desarrollarás habilidades de gestión de datos mediante el uso de diferentes consultas a los sistemas de gestión de bases de datos documentales.

Las preguntas que orientarán esta unidad son:

  • ¿Cómo construir una base de datos mediante un sistema de gestión de bases de datos documentales?
  • ¿Cómo construir consultas (querys) para obtener información de una base de datos documental?
