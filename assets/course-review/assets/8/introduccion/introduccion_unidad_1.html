En esta unidad comprenderemos el concepto de bases de datos documentales, identificando sus ventajas y desventajas respecto a las bases de datos estructuradas o relacionales. Adicionalmente, aprenderemos cómo hacer un diseño de un esquema de datos conceptual y lógico para este tipo de bases de datos.

Las preguntas que orientarán esta unidad son:

  • ¿Qué son las bases de datos documentales?
  • ¿Qué ventajas y desventajas tiene una base de datos documental respecto a una base de datos relacional?
  • ¿cómo realizar un diseño de una base de datos documental?
