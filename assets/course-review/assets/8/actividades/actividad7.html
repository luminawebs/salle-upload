ACTIVIDAD 7: Integración de bases de datos documentales

¿Qué vamos a lograr?

Implementar una conexión de una base de datos documental con un lenguaje de programación.

¿Cómo lo vamos a lograr?

Apreciado estudiante,

A través de esta actividad de manera individual vamos a realizar una conexión de una base de datos documental con un lenguaje de programación. Para iniciar, revisa atentamente los siguientes materiales:

  • Gestión de datos de MongoDB con Python: https://www.mongodb.com/languages/python
  • CRUD Python Mongo: https://www.youtube.com/watch?v=UpsZDGutpZc

Luego de haber revisado el material, procede a resolver el siguiente caso:

Descripción:

Crea una aplicación de gestión de notas que permita a los usuarios realizar las siguientes operaciones:

  • Agregar una Nota: Los usuarios deben poder agregar una nueva nota con un título y un contenido.
  • Ver Todas las Notas: Mostrar todas las notas almacenadas en la base de datos.
  • Buscar Notas por Título: Permitir a los usuarios buscar notas por su título.
  • Eliminar una Nota: Los usuarios deben poder eliminar una nota específica por su título.
  • Actualización de Notas: Permite a los usuarios actualizar el contenido de una nota existente.

Requisitos Técnicos:

  • Uso de Python para escribir el código.
  • Uso de la biblioteca pymongo para conectarte a la base de datos MongoDB.
  • Asegurarse de tener un servidor MongoDB en ejecución localmente o utiliza un servicio de MongoDB en la nube como MongoDB Atlas.

Como producto de la actividad entrega a través de este medio entregar:

  • Video con la explicación de la base de datos creada, incluyendo una explicación de la conexión a las base de datos mediante Python y los comandos usados para cada una de las operaciones realizadas. Tener en cuenta las siguientes recomendaciones:
    • El video no debe superar 10 minutos
    • El video debe compartirse mediante un enlace en un documento .pdf
    • Pueden usarse herramientas como YouTube o Screencast
    • Revisar que el enlace tenga los permisos para poderlo revisar.
