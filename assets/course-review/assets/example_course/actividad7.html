Conocer las diferentes estrategias que permiten optimizar las consultas de una base de datos documental.

Apreciado estudiante,

A través de esta actividad de manera grupal vamos a realizar una consulta sobre las diferentes estrategias de optimización de consultas de una base de datos documental. Para iniciar, revisa atentamente los siguientes materiales:

  • Sarasa, A. (2016). Introducción a las bases de datos NoSQL usando MongoDB. Editorial UOC (disponible aquí).

Luego de haber revisado los materiales, procede a revisar el siguiente guía:

Descripción:

1. Revisar y ejecutar las estrategias propuestas en el capítulo de optimización de  Mongo DB (disponible aquí).

2. Seleccionar dos estrategias de las consultadas (las que más le hayan llamado la atención).

3. Crear una base de datos con un dataset de prueba y ejecutar las estrategias seleccionadas.

Como producto de la actividad entrega a través de este medio entregar:

Video con la explicación de la base de datos creada, incluyendo una explicación de las dos estrategias seleccionadas con los comandos usados para cada una de las operaciones realizadas. Tener en cuenta las siguientes recomendaciones:

  • El video no debe superar 10 minutos
  • Al ser una actividad grupal es importante que todos los miembros del equipo participen en la explicación
  • El video debe compartirse mediante un enlace en un documento .pdf
  • Pueden usarse herramientas como YouTube o Screencast
  • Revisar que el enlace tenga los permisos para poderlo revisar.
