Apreciado estudiante,

A través de esta actividad de manera grupal vamos a crear una base de datos documental. Para iniciar, revisa atentamente los siguientes materiales:

Sarasa, A. (2016). Introducción a las bases de datos NoSQL usando MongoDB. Editorial UOC, capítulo II y III (disponible aquí).

Luego de haber revisado los materiales, procede a revisar el siguiente caso de estudio:

Descripción:

Imagina que estás trabajando en el desarrollo de un sistema de gestión para una biblioteca. La biblioteca necesita un sistema para realizar un seguimiento de sus libros, autores, usuarios y préstamos. Utilizarás MongoDB para almacenar y gestionar la información.

Requisitos:

1. Crea una base de datos llamada "Biblioteca".

2.  Creación de Colecciones:

  • Libros: Para almacenar información sobre los libros, incluyendo el título, autor, género y disponibilidad.
  • Autores: Para almacenar detalles sobre los autores, como el nombre, la fecha de nacimiento y las obras escritas.
  • Usuarios: Para mantener información sobre los usuarios de la biblioteca, como el nombre, la dirección de correo electrónico y los libros prestados.
  • Prestamos: Para realizar un seguimiento de los préstamos, incluyendo el usuario, el libro prestado y la fecha de préstamo.

3. Operaciones CRUD:

  • Realizar inserción de 10 documentos para cada colección creada
  • Realizar actualización de 5 documentos para cada colección creada
  • Realizar eliminación de 2 documentos para cada colección creada
  • Realizar la lectura de datos para cada colección creada.

4. Como producto de la actividad entrega a través de este medio entregar:

  • Video con la explicación de la base de datos creada, incluyendo los diferentes comandos usados para cada una de las operaciones solicitadas en los requisitos.  Tener en cuenta las siguientes recomendaciones:
    • El video no debe superar 10 minutos
    • Al ser una actividad grupal es importante que todos los miembros del equipo participen en la explicación
    • El video debe compartirse mediante un enlace en un documento .pdf
    • Pueden usarse herramientas como YouTube o Screencast
    • Revisar que el enlace tenga los permisos para poderlo revisar.
